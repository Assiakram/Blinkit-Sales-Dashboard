#  Blinkit Sales & Outlet Analysis Dashboard (Power BI)

##  Description
This project presents an interactive Power BI dashboard built to analyze Blinkit sales data and generate meaningful business insights. It highlights key performance metrics such as total sales, number of items, average sales, and customer ratings.

The dashboard provides analysis based on outlet type, outlet size, and item categories to identify high-performing segments and trends for better decision-making.

##  Features
- KPI tracking (Total Sales, Avg Sales, Ratings)
- Sales analysis by item type and outlet
- Outlet size and location insights
- Interactive filters and visuals

##  Tools & Technologies
- Power BI  
- Excel  
- Data Visualization  
- Data Cleaning & Transformation  

##  Insights
- Identified top-performing outlet types  
- Compared low-fat vs regular product sales  
- Analyzed category-wise sales performance
